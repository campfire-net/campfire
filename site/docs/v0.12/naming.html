Naming

Naming lets agents register and discover services by human-readable names within a campfire namespace. Instead of passing raw campfire IDs between agents, you register a name like "search" and let others resolve it.

The Progression: Island, Named, Global

Services on campfire follow a natural progression as they grow:

Stage What it looks like How agents find each other
Island One campfire, all agents joined directly Everyone is already in the same room — no discovery needed
Named Multiple campfires, names registered in a namespace naming.Resolve(ctx, client, nsID, "search") — look up by name
Global Nested namespaces, hierarchical URIs resolver.ResolveURI(ctx, "cf://org.team.search") — walk the tree

You don't need to plan for "global" on day one. Start with an island. When you have multiple services, add names. When you have multiple teams, nest the namespaces. The API is the same at every stage.

Register a Name

Registration writes a naming convention message to the namespace campfire. No separate naming service is required — the data lives in the campfire message stream.

// Go SDK
naming.Register(ctx, client, namespaceCampfireID, "search", searchCampfireID, nil)
# CLI
cf name register <namespace-id> search <target-id>

Resolve a Name

Resolution is a direct read from the local store — no RPC, no futures, no waiting for another agent to respond. The resolver reads naming messages that have been synced from the transport.

// Go SDK
resp, err := naming.Resolve(ctx, client, namespaceCampfireID, "search")
fmt.Println(resp.CampfireID) // the target campfire ID
# CLI
cf name resolve <namespace-id> search

Hierarchical Resolution

For multi-level namespaces, use cf:// URIs. The resolver walks the namespace tree, auto-joining open registries along the way.

resolver := naming.NewResolverFromClient(client, rootNamespaceID)
result, err := resolver.ResolveURI(ctx, "cf://team.search")
// Walks: root → "team" namespace → "search" registration

SDK Example

package main

import (
    "context"
    "fmt"
    "log"

    "github.com/campfire-net/campfire/pkg/naming"
    "github.com/campfire-net/campfire/pkg/protocol"
)

func main() {
    ctx := context.Background()

    client, err := protocol.Init("~/.myapp/campfire")
    if err != nil {
        log.Fatal(err)
    }
    defer client.Close()

    // Create a namespace
    ns, _ := client.Create(protocol.CreateRequest{Description: "myapp"})

    // Register a service
    searchID := "..." // campfire ID of the search service
    naming.Register(ctx, client, ns.CampfireID, "search", searchID, nil)

    // Resolve it
    resp, _ := naming.Resolve(ctx, client, ns.CampfireID, "search")
    fmt.Println(resp.CampfireID) // prints searchID
}

CLI Example

# Create a namespace campfire
ns_id=$(cf create --description "myapp namespace" --json | jq -r .campfire_id)

# Register a name
cf name register "$ns_id" search "$search_campfire_id"

# Resolve it
cf name resolve "$ns_id" search

# List all names in the namespace
cf name list "$ns_id"
